CodingBar 桌機版安裝教學影片｜windows系統

（進入網站的步驟）

（下載安裝之後）

window-1 首先，將下載下來的檔案點擊滑鼠左鍵兩下開啟。

window-2 Windows 系統本身會開啟安全性警告，滑鼠左鍵點選執行。

window-3 接著就可以看到 CodingBar 的安裝畫面，請依照指示安裝，滑鼠左鍵點選下一步。

window-4 請繼續滑鼠左鍵點選下一步。

(請勿更動預設安裝路徑，以免造成程式執行的讀取問題)

window-5 請繼續滑鼠左鍵點選安裝以開始。

window-6 請稍待安裝過程，在過程中請勿按下取消，以免安裝失敗導致程式檔案不完整。

window-7 當安裝完成時，滑鼠左鍵按下完成結束安裝程式。

window-8 桌面應該會出現 CodingBar 程式的捷徑，或是從檔案路徑 C://Program Filse (x86)/Airabbi/CodingBar/，也可以找到程式路徑。滑鼠左鍵點擊兩下 CodingBar.exe 即可開啟程式。

(請勿擅自將程式路徑內的任一檔案移動或刪除，會導致程式檔案不完整。)

window-9 就可以成功開啟 CodingBar 桌機版囉！
