常見問題

Q1：你們的營隊跟一般其他程式營隊有什麼不同嗎？
不同於坊間大部分程式課程或營隊，我們不教「積木式」的程式語言（如：SCRATCH），而直接教現今最熱門、也最容易學習的程式語言 Python 。因為過去我們的經驗，很多孩子玩「積木式」程式語言後，「由奢入儉難」，反而排斥一行一行寫程式，所以我們不教積木式程式語言，而從課程設計來降低孩子學習的障礙，我們將視整體營隊的學習進度，動態調整教學課程內容以達最佳學習品質。
Q2：孩子學要「寫」程式的 Python 會不會太難？
透過 micro:bit 微電腦板學 Python，至今我們還沒遇過學生覺得很難或是無趣的！加上營隊中闖關遊戲、競賽挑戰，反而大家都會覺得時間怎麼過得這麼快喔～如果您還有疑慮，歡迎報名工作坊來體驗： CodingBar 程式入門工作坊
Q3：如果沒有筆電可以幫忙租用嗎？
可以的，營隊期間每台筆電的租金費用為 $2,000元。請在報名時，於備註欄填寫需求。租金將額外收取。
Q4：可以所有營隊都參加嗎？內容會不會重複？
可以的，Python 的基礎語法所有營隊都會教，但應用不同，可以幫助孩子更加融會貫通。
Q5：營隊上完後，如果對程式設計很有興趣，後續有什麼課程嗎？
有的，我們有四階段共 256 小時的課程，您可以選擇實體上課或是線上學習。詳細資訊您可以參閱 CodingBar 青少年課程內容，或透過 LINE : @codingbar 我們將有專人為您說明。
Q6：這兩個營隊怎麼選擇？
A營隊 Python程式探險家活動性比較強，會有一天到戶外透過 micro:bit 尋寶喔！推薦給升五、六年級的孩子。B營隊 Python程式數學家則結合數學問題，讓孩子瞭解數學不只是計算，還有很多生活上的應用需要透過數學來解決，特別推薦給不喜歡數學的孩子，讓他們感受數學的多種應用和樂趣。
